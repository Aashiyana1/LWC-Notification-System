# Cross-Component Notification & Collaboration System

## Features
- Notification Bell with unread count
- Message List
- Detail View
- Lightning Message Service (LMS)
- Apex + Custom Object
- Toast, Alert, Prompt
- Lifecycle methods (connectedCallback, disconnectedCallback)

## Deployment
1. Authorize org
2. Deploy source from VS Code
3. Add components to Lightning App Builder

## Test Data
Create Notification__c records with:
- Title__c
- Message__c
- Status__c (Unread/Read)
